# parents

How many times will I see my parents before they die?

Deployed on https://parents.surge.sh.

## Usage

See `package.json` for details. All tools are installed as development dependencies. Use `yarn`.

### Development

Development:

```
yarn serve
yarn build
```

Deployment:

```
yarn deploy
```
